# Keyword arguments

* -t markers where the text inside is parametrized, default $
* -p --payloads positional payloads e.g "a,b,c,d,e,f" or rockyou.txt 
* -v --verbose
* -m --match math text in the response
* url encode
* auto content-lenght
* redirections
* network error handling
* attack type

# Requirements
* uncurl
* argparse
* python >= 3.8
